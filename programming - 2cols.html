Specular + Diffuse Shading

programming, pixel bender

I implemented a common real-time shading technique. The code has two ways of computing the specular component, a regular cosine diffuse component, options for glossiness and for moving the light in 3D space. Implemented in Adobe Pixel Bender - I found that I can iterate faster with it as opposed to other shader prototyping platforms. Email me for code.
