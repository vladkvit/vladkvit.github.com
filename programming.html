Specular + Diffuse Shading

programming, pixel bender

I implemented a common real-time shading technique. The code has two ways of computing the specular component, a regular cosine diffuse component, options for glossiness and for moving the light in 3D space. Implemented in Adobe Pixel Bender - I found that I can iterate faster with it as opposed to other shader prototyping platforms. Email me for code.

Quadcopter

programming, real-time, embedded

I sponsored and did quite a bit of work to help with my brother's quadcopter. This was his Gr 12 robotics club project, and they ended up winning second place @ World championship. Unfortunately, I don't have any pictures of it flying - I'll post some up later, hopefully.

Here's the code Github page. The basic idea was to try using a tiny x86 board with TCP/IP over WiFi, and a Ground Control Station. We used QGroundControl as the GCS. We were using 500Hz ESCs (brushless motor controllers), a 500Hz AHRS (heading and reference fusion of 3D gyro + 3D magnetometer + 3D accelerometer), and a Maestro USB-PWM adapter. Although using x86 + OS was a bad idea, it still worked.

Software Rasterization

3D, programming, school

As a part of my CS488 Graphics course, I wrote a software implementation of edge rasterization. Features a full projection matrix, screen clipping, frustum culling, etc + object transformation. The 2 RGB gnomons symbolize the model-space and world-space. A line drawing API was given.

Mars Rover Webcam Pano

programming, Matlab, Pixel Bender

I helped out with UW's Mars Rover team. One of the things I did was try to get a merged view of the webcams that were on the rover. Since there wasn't much time before the competition (and I had a huge workload during that semester), I didn't have time to try any fancy approaches (RANSAC), and approximated with an affine transform. This didn't end up being used for the rover, as anything other than the ground would diverge.
